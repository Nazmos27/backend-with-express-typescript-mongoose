# How to run locally

.
.

### Set up the Project

- Clone the project
- cd Ecommerce_RestAPI_with_Mongoose-Typescript
- ts --init
- set "rootDir": "./src", "outDir": "./dist", in the "tsconfig.json" file

### Run the Project

- npm run start:prod
- go to localhost:5000/api/products for products
- go to localhost:5000/api/orders for orders
